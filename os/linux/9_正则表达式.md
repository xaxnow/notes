1.特殊字符
``` 
.*[]^${}\+?|()  
.  任意单个字符 
\*  *前的字符出现0次或多次  
?  ?前的字符出现0或1次  
\+  +前的字符出现1或多次     
\{\}  指定字符出现次数或区间,如e\{1\},或额\{2,5\}  
[afd]	字符组,限定几个特殊字符,即afd中任意一个  
[^afd]	反转字符数组,排除afd    
|  逻辑或,如cat|dog,只要一个匹配就通过  
()  表达式分组,file(name)
```
2.锚字符
```
脱字符^		锁定行首	$	行尾  
```
3.区间
```
[0-9] [a-c] [a-ch-z]
```
4.特殊字符组
``` 
[[:alpha:]] 匹配任意字母字符，不管是大写还是小写  
[[:alnum:]] 匹配任意字母数字字符0~9、A~Z或a~z  
[[:blank:]] 匹配空格或制表符  
[[:digit:]] 匹配0~9之间的数字  
[[:lower:]] 匹配小写字母字符a~z  
[[:print:]] 匹配任意可打印字符  
[[:punct:]] 匹配标点符号  
[[:space:]] 匹配任意空白字符：空格、制表符、NL、FF、VT和CR  
[[:upper:]] 匹配任意大写字母字符A~Z
```
