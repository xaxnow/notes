1.位置参数(命令行参数)
```shell
./test.sh  10 20
#$0,即shell脚本名,$1是10,$2是20,以此类推,超过十个后要这样使用${10}
#  $#获得参数个数,$?上一命令状态,$*和$@返回所有参数
#  $$返回进程id
```

2.移动变量  shift
```
每个参数向左移动一个位置,$3移到$2,把$2处的变量移动到$1中，而$1被删除,$0不变
```
