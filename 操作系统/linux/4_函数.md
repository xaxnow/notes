```shell

name() {
  commands
}
```

3.返回值

$?变量会返回执行的最后一条命令的退出状态码
退出状态码必须是 0~255
默认退出状态码 $?
指定退出状态码 return

4.使用变量

$0 当前脚本文件名
$# 传递给函数或脚本的个数
$*/$@ 传递给函数或脚本所有参数,当被""包含时$*把所有参数当一个整体,$#则当作个体
